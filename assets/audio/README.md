# 音声ファイル一覧

## フォルダ構成

```
audio/
├── bgm/    ← BGM（ループ再生）
└── se/     ← 効果音（1回再生）
```

## スクリプトでの使い方

```
@bgm spring_breeze.mp3   ← BGM再生（ループ）
@bgm stop                ← BGM停止
@se school_bell.mp3      ← SE再生（1回）
```

## 対応フォーマット

MP3 / OGG / WAV（MP3推奨）
