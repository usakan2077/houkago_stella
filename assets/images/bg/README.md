## 新しい背景を追加するには

1. `bg/` フォルダに 1920×1080 JPG を配置
2. `config.js` の `backgrounds` にフォールバック用グラデーションを追加:
   ```js
   park: 'linear-gradient(180deg, #87CEEB 0%, #c8e8a0 100%)',
   ```
3. スクリプトで使用:
   ```
   @scene park fade
   ```

## 注意事項

- `evening_classroom.jpg` と `night_rooftop.jpg` は後方互換のためのコピー
  - `evening_classroom` → `classroom_evening` と同じ画像
  - `night_rooftop` → `rooftop_night` と同じ画像
- スペース付きの元ファイル（`school_gate .jpg` 等）は変換済みのため不要
