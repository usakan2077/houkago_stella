メッセージウィンドウ統合メニュー案

セーブ・ロードなどのコマンドをテキストウィンドウ側へ寄せたときの比較用モックです。 同じ文面で「会話表示 / 地の文表示」と「PC幅 / スマホ幅」を切り替えて見比べられます。

A. 名前レール横並び

常時表示
真昼
Q.SAVE
Q.LOAD
AUTOA
SKIPS

「この高さなら、名前の横にコマンドがいても視線の流れは切れにくいと思う」

夕方の図書室は静かで、ウィンドウの上辺だけがやわらかく光っていた。

▼
良さ: 常設を `SKIP / AUTO / Q.SAVE / Q.LOAD` に絞ると、実用性を残したままかなりすっきりする。
気になる点: `CONFIG` や `TITLE` は別導線に寄せる前提なので、全体設計とセットで考える必要がある。
