# シナリオチェック

作成日: 2026-04-22

## 対象

- `scenarios/chapter1.md` 〜 `scenarios/chapter8.md`
- `scenarios/route_sakura.md`
- `scenarios/route_kotoha.md`
- `scenarios/route_mahiru.md`
- `scenarios/bad_end.md`

## 点検観点

- シナリオ本文の流れに不自然な飛びがないか
- 時間軸、季節、移動元 / 移動先、場所の整合性
- BGM / 効果音 / スチル画像の指定が本文と噛み合っているか
- テキストと演出素材の間に矛盾がないか
- 参照している演出素材が実際に存在するか

## 総評

全体として、共通ルートから各個別ルートへの流れはかなり自然で、時間軸も大きく破綻していない。  
特に以下は安定している。

- 季節進行
  - `chapter1` 新学期
  - `chapter2` 春の深まり
  - `chapter3` GW明け
  - `chapter4` 梅雨
  - `chapter5` 7月のプール回
  - `chapter6` 夏休み
  - `chapter7` 9月
  - `chapter8` 10月の文化祭
- さくら / ことは / まひるの個別ルートへ繋がる伏線配置
- 主人公の過去の傷と各ヒロインの悩みの接続
- 場所移動の大枠

致命的なシナリオ破綻は見当たらなかった。  
一方で、明確な要修正が1件、軽微な演出違和感が数件ある。

## 要修正

### 1. `route_kotoha.md` で存在しない効果音を参照している

対象:

- `scenarios/route_kotoha.md:565`
- `scenarios/route_kotoha.md:598`

内容:

- `@se wind_leaves.mp3` を使用しているが、`assets/audio/se/` に `wind_leaves.mp3` が存在しない。

影響:

- 実機では該当箇所でSEが鳴らない。
- ログやコンソール上で 404 / 読み込み失敗が出る可能性がある。

補足:

- この場面は秋の通学路なので、演出的には葉擦れSE自体は合っている。
- 対応としては以下のどちらか。
  - `wind_leaves.mp3` を追加する
  - 既存SEに差し替える

### 2. `route_sakura.md` の体育館シーンは、場所のつながりを実機で一度確認したい

対象:

- `scenarios/route_sakura.md:1256` 以降

内容:

- 体育館裏での会話後、本文で
  - `一緒に帰り始めた。`
  - `廊下を抜けて、階段へ。`
  とつなぎ、`@scene staircase_evening fade` に切り替わる。

確認ポイント:

- フェードの体感が自然か
- 体育館裏 → 校舎内階段への遷移がプレイ中に唐突に見えないか

優先度:

- 低

## 問題なしだった点

### 共通ルート

- `chapter1` 〜 `chapter8` の季節進行は自然
- ヒロインごとの導入位置と接触頻度は概ね適切
- 学校内の移動
  - 教室
  - 廊下
  - 図書室
  - 屋上
  - 音楽準備室
  - 体育館
  の流れに大きな矛盾なし
- 文化祭までの積み上げは安定している

### ことはルート

- 文化祭の失敗から、図書室 / 音楽準備室 / 合唱コンクールへの再起の流れは自然
- スチルの入りどころ
  - `kotoha_music_door`
  - `kotoha_library_headphones`
  - `kotoha_piano_trembling`
  - `kotoha_one_note`
  - `kotoha_performance1`
  - `kotoha_performance2`
  は文脈と噛み合っている
- 時間経過
  - 文化祭
  - 数日後
  - 一ヶ月後の合唱コンクール
  の流れに不自然さなし

### まひるルート

- 十月初旬 → 十月下旬 → 命日前夜 → 命日の朝、という時間軸は通っている
- 屋上関連の場所描写と心理描写の一貫性が強い
- 主人公が「走れない」状態から再び走るまでの感情線も自然
- `bad_end` と比較したときの分岐意味も明確

### さくらルート

- 家庭事情、部活、父の入院、母への返信、後輩との関わりが一本の線で繋がっている
- 体育館の崩れるシーンは、現行テキストでは「立ち姿ベース」にかなり揃っている
- グッドエンドへの導線も無理がない

### `bad_end.md`

- 文化祭当日の取りこぼしが、3人それぞれの断片に落ちる構成は分かりやすい
- 主人公の「また間に合わなかった」感覚が共通ルートから自然に接続されている
- まひるルートの対比としても機能している

## まとめ

現時点で修正必須なのは、`route_kotoha.md` の `wind_leaves.mp3` 参照切れ。  
それ以外は、演出の温度差や画面遷移の見え方として確認しておきたい項目が中心。

総合的には、

- 話の流れ: 良好
- 時間軸: 良好
- 場所の整合性: 良好
- スチル挿入: 概ね良好
- BGM: 概ね良好
- SE: 1件要修正、数件確認推奨

という判定。
